Usage
==============================
### Simple router regist
``` swift
	let urlMappings = ["home":"ListViewController",
					   "vc00":"ViewController",
					   "vc01":"ViewController01",
					   "vc02":"ViewController02",
					   "vc03":"ViewController03",
					   "vc04":"ViewController04"]
	FBRouter.router().registURLMapping(urlmappings: urlMappings)
```
